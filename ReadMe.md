**內容:**

>從一副撲克牌中取出12張並複製使其變成兩張，將24張牌翻面使其背面朝上，並隨機牌排成6*4 的矩陣，第一次會先由電腦翻開所有牌後再蓋上，之後由玩家一次翻兩張牌，若連續翻起的兩張牌均相同，則可取走此兩張牌，否則電腦會自動蓋回，直到所有牌均取完為止。
